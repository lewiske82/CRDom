# CRDom
AI CRM System

## Unified Dev Core 3.0 – Hybrid Memory
Ez a repo egy mini CRM alapot és egy multi-agent vezérlő magot tartalmaz, amely:
- ELI10 kérdezési stílusban bekéri a hiányzó paramétereket.
- Engedély-alapú memóriát és többszintű audit naplót vezet.
- Oracle-hibrid offline/online üzemmódra készül.

Indítási példa:
```bash
python3 codex_init.py
```

További részletek: `docs/architecture.md`, `docs/api_schema.md`, `docs/short_prompt.md`.
